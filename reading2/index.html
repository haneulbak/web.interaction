WHAT IS CODE?

link

Summary

There are million ways to define and understand what code is. A coder's understanding of code is as valid as the next's.
But it is important to look into the relations between various elements of coding, such as computer, algorithm, software, or language.

Quotes

  • “Algorithm” is a word writers invoke to sound smart about technology.
    Journalists tend to talk about “Facebook’s algorithm” or a “Google algorithm,” which is usually inaccurate.
    They mean “software.”
  • A programming language is a system for encoding, naming, and organizing algorithms for reuse and application.
    It’s an algorithm management system. This is why, despite the hype, it’s silly to say Facebook has an algorithm.
    An algorithm can be translated into a function, and that function can be called (run) when software is executed..
